# Введение

Инструкция по настройке интеграции магазина на Yandex.Market с сервисом доставки от СДЭК

:::info

Данная интеграция предназначена для автоматического создания накладных СДЭК для заказов, оформленных по модели DBS.
Подробнее про модель продаж DBS можно прочитать в [инструкции](https://yandex.ru/support/marketplace/ru/orders/dbs/).

:::

Основные особенности интеграции:
- Автоматическое создание накладных СДЭК для заказов на Yandex.Market без необходимости ручного переноса данных
- Синхронизация статуса доставки заказов между Yandex.Market и СДЭК
- Прогнозирование размера упаковки на основе товаров заказа и подбор оптимального тарифа для отправки

<details>
    <summary>Принцип работы интеграции</summary>

    После настройки, система каждые 20 минут проверяет наличие новых заказов на Yandex.Market.

    При обнаружении нового заказа, автоматически создается накладная в СДЭК.

    Накладная появляется в [личном кабинете](https://lk.cdek.ru), откуда ее можно распечатать и наклеить на посылку для
    передачи на доставку.
</details>

# Установка и настройка

Для работы интеграции, не требуется установка дополнительного ПО. Вся настройка происходит через личный кабинет СДЭК.

Для этого необходимо перейти на страницу [настроек интеграции](https://lk.cdek.ru/integration) и выбрать режим **DBS**.

<Image img={require('./img/start.png')} />

Далее необходимо сгенерировать токен Yandex.Market и ввести его в настройках интеграции.
Инструкцию по генерации токена можно прочитать на [отдельной странице](https://yandex.ru/dev/market/partner-api/doc/ru/concepts/api-key).
